# Files Upload & Co
- Upload implemented in `PHP`, in `data.sender.php`.
- Client samples in the `sample.clients` folder
- Run a sample
  - From the `uploads` folder, start `php -S localhost:8000`
  - Then from a browser, reach <http://localhost:8000/sample.clients/upload.html>, or <http://localhost:8000/sample.clients/programmatic.upload.html>
  - Do look in the html code, to show/hide some elements, change the location of the php service to invoke (`action` attribute of the forms), etc...
