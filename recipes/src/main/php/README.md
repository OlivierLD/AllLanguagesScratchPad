# Recipes DB

PHP and SQLite.
Also shows how to upload and download PDFs in a BLOG.


# How-to
1 - Start the php server from the `php` directory:
```
php $ php -S localhost:8000
```
2 - From a browser, reach `http://localhost:8000/index.html`

More [doc](../../../../../AllLanguagesScratchPad/src/main/java/SQLITE.md) on SQLite.
